# UI-session
using Local Storage and created user interface
Designed a login and registration Ul using HTML and CSS.
Captured user credentials using JavaScript form handling.
Stored user data securely (for demo purpose) in LocalStorage.
Validated login credentials by comparing stored values.
Displayed success and error messages dynamically.
Implemented logout functionality by clearing session data.
Restricted access to a protected page without login.
Redirected users based on authentication status.
